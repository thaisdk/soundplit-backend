# Soundplit 

A RESTful API built with Django framework.

## Como rodar

Instale os requerimentos:
```bash
pip install -r requirements.txt 
```

Crie um arquivo ```.env``` baseado no _.env-example_

Rode o projeto:
```bash
make run
```

### Opcional

Criar ambiente virtual:
```bash
python -m venv venv
```

## Comandos

### Criar um app

`make startapp [app_name]`

### Criar migrações

`make migrate`
